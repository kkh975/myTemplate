Swipe Scroll

Option

isSnap Boolean false 스냅 기능 활성화
isCssAllInline Boolean true Recommend!! 내장 스타일 설치
initIndex Number 0 초기화 위치
installCallback Function 설치 후 callback
beforeCallback Function 액션 전 callback
activeCallback Function 액션 후 callback

Method

disable 비활성
enable 활성
destroy 제거

Use

기본

			$('.mySwipeScroll').mySwipeScroll();

옵션

			$('.mySwipeScroll').mySwipeScroll({
				isForce: true,
				activeClassName: 'active',
				installCallback: function() {
					console.log( 'install' )
				},
				openCallback: function() {
					console.log( 'active' )	
				}
			});

Method

			$('.mySwipeScroll').mySwipeScroll('disable')
			$('.mySwipeScroll').mySwipeScroll('enable')
			$('.mySwipeScroll').mySwipeScroll('destroy')
