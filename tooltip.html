Tooltip

Option

isForce Boolean false 강제 모드 활성화 여부(지원 브라우저에서는 설치 안됨)
activeClassName String my_tooltip_active 활성시 class 이름
installCallback Function 설치 후 callback
activeCallback Function 설치 후 callback

Method

disable 비활성
enable 활성
destroy 제거

Use

기본

			$('.myTooltip').myTooltip();

옵션

			$('.myTooltip').myTooltip({
				isForce: true,
				activeClassName: 'active',
				installCallback: function() {
					console.log( 'install' )
				},
				activeCallback: function() {
					console.log( 'active' )	
				}
			});

Method

			$('.myTooltip').myTooltip('disable')
			$('.myTooltip').myTooltip('enable')
			$('.myTooltip').myTooltip('destroy')
