Modal

Option

isForce Boolean false 강제 모드 활성화 여부(지원 브라우저에서는 설치 안됨)
activeClassName String my_modal_active 활성시 class 이름
installCallback Function 설치 후 callback
openCallback Function 설치 후 callback

Method

disable 비활성
enable 활성
destroy 제거

Use

기본

			$('.myModal').myModal();

옵션

			$('.myModal').myModal({
				isForce: true,
				activeClassName: 'active',
				installCallback: function() {
					console.log( 'install' )
				},
				openCallback: function() {
					console.log( 'active' )	
				}
			});

Method

			$('.myModal').myModal('disable')
			$('.myModal').myModal('enable')
			$('.myModal').myModal('destroy')
